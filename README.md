# Sales Data Analysis

## 📌 Project Overview
This project analyzes a sales dataset containing 1000 records to extract meaningful business insights.

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Key Analysis
- Sales by Region
- Sales by Product Category
- Top Sales Representatives
- Monthly Sales Trends
- Profit Analysis

## 📈 Key Insights
- Identified top-performing regions and product categories
- Analyzed customer types and sales channels
- Calculated profit using cost and price data

## 📁 Files
- sales_analysis.ipynb → Jupyter Notebook with analysis
- sales_data.csv → Dataset used

## 🚀 Outcome
Performed data cleaning, feature engineering, and visualization to support data-driven decision-making.
